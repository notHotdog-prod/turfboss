FAQ

Common Questions

Can artificial turf go on a flat roof?+

Yes, with proper engineering. We install lightweight systems specifically designed for rooftops, protecting the membrane while providing proper drainage integration. We assess each rooftop's load capacity before installation.

Is it slippery when wet around pools?+

No — our pool deck turf features non-slip backing and drains quickly. It's actually safer than many traditional pool deck surfaces like smooth concrete or tile.

Can turf go on an existing wood deck?+

Yes. We install turf over existing wood, composite, and concrete surfaces using appropriate adhesive systems and drainage considerations for each surface type.

Does chlorine damage the turf?+

Quality UV-stabilized pool turf is resistant to chlorine and other pool chemicals. We select products specifically rated for pool environments.
